# Project Overview

This project is aimed at creating an innovative solution to improve productivity in various domains. It offers a streamlined approach to manage tasks effectively.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/duongnguyen321/tradot.git
   ```
2. Navigate into the project directory:
   ```bash
   cd tradot
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
